Benvenuto in Adventure Studio

Crea avventure dark fantasy compatibili con .txt-Master in pochi passi.

  • 1 Premi Nuovo evento o usa il tasto N per aggiungere nodi alla mappa del flusso.
  • 2 Clicca un nodo per aprire l'editor: aggiungi testo, scelte, mostri e loot.
  • 3 Trascina il pallino arancione a destra di un nodo verso un altro per collegare le scene.
  • 4 Usa Salva avventura (o Ctrl+S) per salvare localmente nel browser in qualsiasi momento.
  • 5 Premi Esporta JSON per ottenere il file compatibile con l'app Android.

Puoi riaprire questo tutorial dal tasto Tutorial in basso nella sidebar.
